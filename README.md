# StarFighter 🚀🌌

O **StarFighter** é um jogo de combate espacial interativo do género *shoot 'em up* (as famosas "navezinhas"), desenvolvido na **Godot Engine**. No comando de uma nave estelar avançada, o jogador deve enfrentar ondas de inimigos, desviar-se de obstáculos cósmicos e sobreviver no espaço profundo.

---

## 🚀 Funcionalidades

- **Mecânica de Combate:** Sistema de tiros dinâmico com deteção de colisão e eliminação de inimigos.
- **Movimentação Fluida:** Controlos responsivos da nave para desviar-se de ameaças em tempo real.
- **Sistema de Pontuação:** Registo de progresso ou destruição para desafiar as pontuações mais altas.
- **Interface de Utilizador (UI):** Menus e HUD integrados para exibir informações vitais (vida, pontos, etc.).

---

## 🛠️ Tecnologias & Stack

O jogo foi totalmente concebido utilizando o ecossistema da Godot:

- **Motor de Jogo:** [Godot Engine](https://godotengine.org/) (Versão estável)
- **Linguagem de Programação:** [GDScript](https://docs.godotengine.org/pt-br/stable/tutorials/scripting/gdscript/gdscript_basics.html) (Lógica de movimentação, IA dos inimigos e colisões)
- **Componentes:** Nós nativos da Godot para Física 2D (KinematicBody2D/CharacterBody2D, Area2D), Partículas e Sprites.
